# @bisonai/sbt-js

## Prerequisites

Generate mnemonic.

```shell
npx mnemonics
```

Create `.env` from `.env.example`

```shell
cp .env.example .env
```

to run on baobab fill in generated mnemonic to `MNEMONIC0` and `MNEMONIC1` environment variable.

## Installation

```shell
yarn install
```

## Build

```shell
yarn build
```

## Test

```shell
yarn test
```
